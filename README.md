# Structura

> Transform Documents into Interactive Learning.

Structura is an AI-powered learning platform that converts PDFs into structured courses with AI-generated lessons, quizzes, progress tracking, and an AI tutor.

## Tech Stack

- Next.js
- FastAPI
- PostgreSQL
- ChromaDB
- Groq
- Clerk
